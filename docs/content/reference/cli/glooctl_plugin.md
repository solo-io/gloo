## glooctl plugin

Commands for interacting with glooctl plugins

### Synopsis

Commands for interacting with glooctl plugins. Glooctl plugins are arbitrary binary executables in your path with the prefix 'glooctl-'.

```
glooctl plugin [flags]
```
